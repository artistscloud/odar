┌──────────────────────────────────┐
│ BASIC OPERATION                  │
├──────────────────────────────────┤
│ Power On: Hold 2s               │
│ Power Off: Hold 3s              │
├──────────────────────────────────┤
│ BUTTON FUNCTIONS                 │
│ ◯ MENU                          │

│ ◯ SELECT                        │

## Card 2: Measurement Mode
```
┌──────────────────────────────────┐
│ MEASUREMENT MODE                 │
├──────────────────────────────────┤
│ STANDARD MEASUREMENT             │
│ 1. Press SELECT to start        │
│ 2. Hold steady                  │
│ 3. Wait for reading (1-3s)      │
│ 4. Press SELECT to stop         │
├──────────────────────────────────┤
│ TEMPERATURE CYCLE                │
│ 1. MENU → Temp Cycle           │
│ 2. Wait for completion         │
│    (10°C → 40°C)               │
│ 3. Review results              │
├──────────────────────────────────┤
│ RANGING MODE                     │
│ 1. MENU → Ranging              │
│ 2. Rotate 360°                 │
│ 3. Follow direction indicator   │
│ 4. Approach source             │
└──────────────────────────────────┘
```

## Card 4: Common Error Recovery
```
┌──────────────────────────────────┐
│ ERROR RECOVERY                   │
├──────────────────────────────────┤
│ E001: BATTERY LOW               │
│ → Connect charger               │
│ → Wait for 10% charge          │
├──────────────────────────────────┤
│ E002: SENSOR ERROR              │
│ → Clean sensor intake          │
│ → Wait 5 minutes               │
│ → Restart device               │
├──────────────────────────────────┤
│ E003: TEMP ERROR                │
│ → Check ambient temp           │
│ → Allow stabilization          │
├──────────────────────────────────┤
│ E004: RANGING ERROR             │
│ → Clear sensor path            │
│ → Reduce distance (<4m)        │
└──────────────────────────────────┘
```
